# PetMap

Plataforma desenvolvida como projeto acadêmico para busca e divulgação de hotéis para pets, facilitando que donos encontrem hospedagens e que proprietários de hotéis tenham uma plataforma para divulgar seus estabelecimentos.
- Conta com busca de hotéis por nome ou endereço, mapa interativo com geolocalização do usuário, cadastro e autenticação de usuários, cadastro de hotéis, exibição de informações (contato, redes sociais, pets aceitos e avaliações) e integração com banco de dados para armazenamento e consulta das informações

## Tecnologias:
- HTML5
- CSS3
- JavaScript
- Firebase (Authentication e  Cloud Firestore)

## APIs e bibliotecas:
- Geolocation para obter a localização atual do usuário
- Leaflet.js para renderização do mapa e dos marcadores
- OpenStreetMap para fornecer os dados e as imagens do mapa.
- Firebase Authentication para autenticação de usuários.
- Cloud Firestore para o armazenamento e consulta dos dados dos hotéis
